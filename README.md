# Massachusetts Utility Layers: Analytical Dashboard

A high-performance GIS dashboard for Platanus Labs, visualizing spatial utility data (GSEP & Gas Leaks) alongside real-time statistical analytics.

## Features
- **Spatial Maps**: Interactive ArcGIS mapping of statewide utility infrastructure.
- **Data Analytics**: Live reactive charts (Chart.js) showing Infrastructure Allocation, Project Timelines, and Safety Status.
- **Dynamic Filters**: Real-time synchronization between map visibility and statistical reporting.
- **Glassmorphic UI**: Premium dark-mode aesthetic with a high-density 'Tight Layout' for pro-users.

## Tech Stack
- **ArcGIS Maps SDK for JavaScript (v4.31)**
- **Chart.js**
- **Vanilla CSS / Semantic HTML**
